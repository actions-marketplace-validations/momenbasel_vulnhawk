# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in VulnHawk itself, please report it responsibly.

**Do not open a public issue.** Instead, use [GitHub Security Advisories](https://github.com/momenbasel/vulnhawk/security/advisories/new) to report it privately.

You will receive a response within 48 hours.

## Scope

- Vulnerabilities in VulnHawk's code (CLI, scanner, reporters, action)
- Prompt injection risks in LLM interaction
- Dependency vulnerabilities in direct dependencies

## Out of Scope

- Vulnerabilities found *by* VulnHawk in other codebases (that's the tool working as intended)
- Issues with third-party LLM providers (Anthropic, OpenAI, Ollama)
